# CanSat

Sistema CanSat desarrollado para adquisición de datos, telemetría y control mediante microcontroladores ESP32 y sensores ambientales.

Descripción

Este proyecto contiene el código utilizado para el funcionamiento de un CanSat, incluyendo:

Lectura de sensores 
Comunicación inalámbrica 
Captura de imágenes 
Almacenamiento y procesamiento de datos 
Control del sistema durante el descenso

El objetivo es simular las funciones básicas de un satélite dentro del volumen de una lata.

*Hardware utilizado*

- ESP32-C3 / ESP32-CAM
- Sensores ambientales
- Módulo de comunicación NRF24l01
- Batería 9v
- Estructura CanSat

*Librerías*

- RF24 (by TMRh20)
- Adafruit BMP3XX
- Adafruit BNO055
- Adafruit Unified Sensor
- ESP32 board package (para ESP32-CAM)
- ESP32-C3 board package
